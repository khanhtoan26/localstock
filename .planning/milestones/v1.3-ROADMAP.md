## Milestone Summary

**Key Decisions:**

- Source Sans 3 variable font via next/font/google — FOUC-free, Vietnamese subset, no paid hosting
- Warm neutral palette (near-black primary) replacing all blue — Claude Desktop aesthetic
- FloatingSidebar: icon rail (w-14) always visible + overlay (w-60) with 180ms CSS transform — avoids push-content layout complexity
- nuqs removed from TBL-04 — URL search persistence dropped in favor of local useState after verification revealed nuqs added complexity without clear benefit
- GET /api/market/summary uses MAX(date) not date.today() — robust across weekends and market holidays
- MarketSummaryCards uses staleTime: 30min — aligns with daily crawl rhythm, avoids unnecessary refetches

**Issues Resolved:**

- Blue hardcoded in chart-colors.ts, grade badges, score breakdown — fully replaced with neutral stone palette
- Sort on numeric columns was alphabetical — fixed with pure sort-comparator.ts module
- Market session bar hydration mismatch — fixed with useSyncExternalStore SSR guard
- Silent zero-to-null coercion on total_volume — removed or None truthy coercion
- Hardcoded English strings in MarketSummaryCards — extracted to i18n keys (EN + VI)

**Issues Deferred:**

- TBL-04 URL search persistence — dropped; search is session-local only
- LAY-05 Sidebar keyboard shortcut (Cmd+B)
- LAY-06 Mobile responsive sidebar

**Technical Debt Incurred:**

- Phase 07 UAT (9 pending scenarios) and Phase 09 VERIFICATION (human_needed) — carried from v1.1, not addressed in v1.3
- nuqs installed then removed — package.json was temporarily modified (nuqs@^2.8.9 added then removed)
