1. **Create `src/localstock/macro/crawler.py` — MacroCrawler:**
   - VCB XML endpoint: `https://portal.vietcombank.com.vn/Usercontrols/TVPortal.TyGia/pXML.aspx`
   - `async def fetch_exchange_rate(self) -> dict | None`: GET the XML, parse with BeautifulSoup (features="xml" or "lxml-xml"), find Exrate element where CurrencyCode="USD", extract Buy/Sell rates, return `{"value": float(sell_rate), "source": "vcb", "indicator_type": "exchange_rate_usd_vnd", "period": today_str, "recorded_at": today}`.
   - Compute trend by comparing with previous value: pass previous_value as optional parameter. If previous exists: value > previous → "rising" (vnd_weakening), value < previous → "falling" (vnd_strengthening), else "stable".
   - Wrap in try/except. On any error (network, parse), log warning and return None. Use httpx.AsyncClient with timeout=15.
   - Do NOT use tenacity retry here — this is a single daily fetch; if it fails, manual entry is the fallback.
   - `async def determine_macro_conditions(self, indicators: list) -> dict[str, str]`: Takes list of MacroIndicator objects, returns dict like `{"interest_rate": "rising", "exchange_rate": "stable"}`. For each indicator_type, use the `trend` field if available, else compare current value to a reference threshold.

2. **Create `src/localstock/macro/impact.py` — MACRO_SECTOR_IMPACT rules:**
   - Define `MACRO_SECTOR_IMPACT: dict[str, dict[str, float]]` — 8 conditions × 20 sectors.
   - Conditions: `interest_rate_rising`, `interest_rate_falling`, `vnd_weakening`, `vnd_strengthening`, `cpi_rising`, `cpi_falling`, `gdp_growing`, `gdp_slowing`.
   - Use impact multipliers from research §Macro-to-Sector Impact Scoring table. Fill in all 20 sectors for each condition. OTHER sector always gets 0.0.
   - `def get_macro_impact(sector_code: str, macro_conditions: dict[str, str]) -> float`: Look up each active condition (e.g., interest_rate=rising → key="interest_rate_rising"), sum impact multipliers for the sector across all active conditions, clamp to [-1.0, +1.0]. Return 0.0 if sector is "OTHER" or not found.

3. **Create `src/localstock/macro/scorer.py` — normalize_macro_score():**
   - `def normalize_macro_score(sector_code: str | None, macro_conditions: dict[str, str]) -> float`: Returns 0-100 score. 50 = neutral.
   - Call `get_macro_impact(sector_code, macro_conditions)` to get aggregate impact (-1.0 to +1.0).
   - Map: impact of +1.0 → score 100, impact of 0.0 → score 50, impact of -1.0 → score 0. Formula: `score = 50 + (impact * 50)`. Clamp to [0, 100].
   - If sector_code is None or macro_conditions is empty dict → return 50.0.

4. **Create tests:**
   - `tests/test_macro/test_crawler.py`: Mock httpx response with sample VCB XML. Test parse success, connection error, malformed XML. Sample VCB XML format: `<ExrateList><DateTime>...</DateTime><Exrate CurrencyCode="USD" CurrencyName="..." Buy="25450" Transfer="25480" Sell="25850"/></ExrateList>`
   - `tests/test_macro/test_impact.py`: Test MACRO_SECTOR_IMPACT structure (all conditions have all sectors), test get_macro_impact for known cases (BANKING + rate rising = positive, REAL_ESTATE + rate rising = negative).
   - `tests/test_macro/test_scorer.py`: Test normalize_macro_score for favorable, unfavorable, neutral, no data, None sector cases. Test output range [0, 100].
