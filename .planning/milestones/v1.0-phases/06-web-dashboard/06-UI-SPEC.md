| md | 16px | Default card padding, section spacing, sidebar section padding |

## Typography

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Body | 14px | 400 (regular) | 1.5 | Table cells, report text, card content, general prose |
| Label | 12px | 400 (regular) | 1.4 | Table column headers, metadata labels, chart axis labels, muted captions |
| Heading | 20px | 600 (semibold) | 1.2 | Page titles ("Xếp Hạng Cổ Phiếu"), section headers ("Phân Tích Kỹ Thuật") |
| Display | 28px | 600 (semibold) | 1.1 | Stock symbol on detail page, hero metric values |

**Rationale:** 14px body is standard for data-dense financial dashboards (Bloomberg, Simplize). 12px labels keep column headers compact for wide tables with 8+ columns. Monospace (`font-mono`) for all numeric score/price values. Two weights only: 400 (regular) for body text and labels, 600 (semibold) for headings and display — size difference alone distinguishes Label from Body.

## Page Contracts

### Rankings Page (`/rankings` — DASH-01)

**Focal point:** DataTable is the primary visual anchor — fills the entire content area width.

**Layout:** Single DataTable filling content area width.

**Table columns (left to right):**

| # | Column Header | Field | Width | Format |
|---|---------------|-------|-------|--------|
| 1 | # | rank | 50px | Integer |
| 2 | Mã CK | symbol | 80px | Link to `/stock/{symbol}`, `text-primary font-medium` |
| 3 | Điểm | total_score | 70px | `font-mono`, 1 decimal (e.g., "72.5") |
| 4 | Hạng | grade | 60px | GradeBadge component (A/B/C/D/F) |
| 5 | Kỹ Thuật | technical_score | 80px | `font-mono`, 1 decimal, `—` if null |
| 6 | Cơ Bản | fundamental_score | 80px | `font-mono`, 1 decimal, `—` if null |
| 7 | Tin Tức | sentiment_score | 80px | `font-mono`, 1 decimal, `—` if null |
| 8 | Vĩ Mô | macro_score | 80px | `font-mono`, 1 decimal, `—` if null |

**Interactions:**
- All columns sortable (click header to toggle asc/desc)
- Default sort: `total_score` descending
- Row click navigates to `/stock/{symbol}`
- Row hover: `bg-muted/50` highlight

**States:**
- Loading: Skeleton rows (8 rows × 8 columns, shimmer animation)
- Empty: Empty state component (see Copywriting)
- Error: Error state component (see Copywriting)

### Stock Detail Page (`/stock/[symbol]` — DASH-02)

**Focal point:** Candlestick price chart (400px) is the primary visual anchor — largest element, first scrollable content after header.

**Layout (top to bottom):**

1. **Header bar** — Symbol (Display size, 28px semibold) + grade badge + score + back link
2. **Price chart** — Candlestick + volume + indicator overlays (400px height)
3. **Timeframe selector** — Button group: 1T (1 month), 3T, 6T, 1N (1 year), 2N
4. **Sub-panels** — MACD panel (152px) + RSI panel (152px), stacked vertically
5. **AI Report card** — Full Vietnamese analysis report in a card
6. **Score breakdown card** — Individual dimension scores in a grid

**Chart type (Agent's Discretion — D-07):** Candlestick + volume histogram overlay on same chart.

**Timeframe options (Agent's Discretion — D-09):**

| Button Label | Days | Description |
|-------------|------|-------------|
| 1T | 30 | 1 tháng (1 month) |
| 3T | 90 | 3 tháng |
| 6T | 180 | 6 tháng |
| 1N | 365 | 1 năm (1 year) |
| 2N | 730 | 2 năm |

Default: **1N** (1 year — shows enough context for trend analysis).

**Indicator toggles:** Not needed for v1. All indicators display by default per D-08: SMA/EMA/BB on main chart, MACD/RSI on sub-panels.

**AI Report rendering:**
- Card with heading "Báo Cáo AI"
- Report text rendered as plain text (not HTML) — paragraphs separated by whitespace
- `text-sm` (14px), `leading-relaxed` (line-height 1.625) for readability
- If report is null: show "Chưa có báo cáo — chạy pipeline để tạo báo cáo AI"

### Market Overview Page (`/market` — DASH-03)

**Focal point:** Macro indicator cards (2×2 grid) are the primary visual anchor — first content block, immediately communicates market state.

**Layout (top to bottom):**

1. **Macro indicator cards** — Grid of 4 cards (2×2 on desktop)
2. **Sector performance table** — Ranked by average score

**Macro cards (each card):**

| Card | Label | Value Format | Source |
|------|-------|-------------|--------|
| Lãi Suất SBV | `{value}%` | 1 decimal percent | GET /api/macro/latest |
| Tỷ Giá USD/VND | `{value}` | `Intl.NumberFormat('vi-VN')` | GET /api/macro/latest |
| CPI | `{value}%` | 1 decimal percent | GET /api/macro/latest |
| GDP | `{value}%` | 1 decimal percent | GET /api/macro/latest |

**Card design:** `bg-card border border-border rounded-lg p-4`. Label in `text-muted-foreground text-xs`, value in `text-foreground text-xl font-semibold font-mono`.

**Sector performance table columns:**

| Column | Header | Format |
|--------|--------|--------|
| 1 | Ngành | Sector name (Vietnamese) |
| 2 | Điểm TB | Average score, 1 decimal |
| 3 | Số Mã | Stock count, integer |
| 4 | Hạng | GradeBadge based on avg score |

## Interaction States

### Loading

- **Table:** 8 skeleton rows, each cell a `Skeleton` block matching column width
- **Chart:** Container div at 400px height with centered `Skeleton` rectangle
- **Cards:** Card outline with `Skeleton` blocks for label + value
- **Transition:** Skeleton → content, no fade animation (instant swap)

### Empty

- **Layout:** Centered vertically in content area
- **Icon:** `BarChart3` (lucide) at 48px, `text-muted-foreground`
- **Heading:** `text-lg font-semibold text-foreground`
- **Body:** `text-sm text-muted-foreground mt-2`
- **CTA (if applicable):** `Button variant="default"` below body with `mt-4`

### Error

- **Layout:** Same centered layout as empty state
- **Icon:** `AlertCircle` (lucide) at 48px, `text-red-400`
- **Heading:** `text-lg font-semibold text-foreground`
- **Body:** `text-sm text-muted-foreground mt-2`
- **No retry button** — user refreshes page (personal tool, keep simple)

### Hover

- **Table row:** `bg-muted/50` on hover, `cursor-pointer`
- **Sidebar nav item:** `bg-muted` on hover (non-active items)
- **Stock symbol link:** `hover:underline`
- **Timeframe button:** Same as shadcn Button hover default

### Active

- **Sidebar nav item:** `bg-primary/10 text-primary` when route matches
- **Timeframe button:** `variant="default"` (filled) for active, `variant="outline"` for inactive

## Number Formatting Rules

| Data Type | Format | Example |
|-----------|--------|---------|
| Composite score | 1 decimal, monospace | `72.5` |
| Sub-scores | 1 decimal, monospace, `—` if null | `65.3` or `—` |
| Stock price (VND) | `Intl.NumberFormat('vi-VN')`, no decimals | `23.450` (VND uses dots) |
| Volume | Abbreviated with suffix | `1,2M` or `523K` |
| Market cap | Billions suffix | `12.345,6 tỷ` |
| Percentage | 1 decimal + `%` | `12,5%` |
| Macro rates | 1 decimal + `%` | `4,5%` |
| Exchange rate | `Intl.NumberFormat('vi-VN')` | `25.432` |
| Rank | Integer, no formatting | `1`, `2`, `15` |
