<objective>
Add the `_step_timer(step_name, run)` async context manager to `Pipeline` and
wrap the existing `run_full` body so that:

- `crawl` = current Steps 1-7 (listings + 4 crawlers + storage) ŌĆö REAL
  duration captured per Q-3
- `analyze` = `_apply_price_adjustments()` ŌĆö REAL duration captured per Q-3
- `score`, `report` = explicit `None` on the PipelineRun row ŌĆö placeholders
  until AutomationService integration in a future phase

The context manager:
1. Stamps `t0 = time.perf_counter()` on enter
2. Computes `duration_ms` in `finally` (so it runs on exception too)
3. Writes `setattr(run, f"{step_name}_duration_ms", duration_ms)` BEFORE re-raise
4. Emits `op_duration_seconds.labels("pipeline","step",step_name,outcome).observe(elapsed)`
5. On exception sets `outcome="fail"` THEN re-raises

This plan does NOT use the `@observe` decorator (24-01) on `run_full` directly ŌĆö
the context manager is required because `@observe` cannot write to the
`PipelineRun` row (it only emits the histogram). Both the column write AND the
metric observation are the goal here.

    ```python
        @asynccontextmanager
        async def _step_timer(self, step_name: str, run):
            """Phase 24 D-08 ŌĆö time a pipeline step.

            Records `setattr(run, f"{step_name}_duration_ms", duration_ms)` AND
            emits localstock_op_duration_seconds (labels: pipeline.step.<name>.<outcome>)
            even on exception. Re-raises original exception unchanged.

            Pitfall 7: ordering is try / yield / except (set fail outcome,
            re-raise) / finally (record duration + observe metric).
            """
            t0 = time.perf_counter()
            outcome = "success"
            try:
                yield
            except Exception:
                outcome = "fail"
                raise
            finally:
                elapsed = time.perf_counter() - t0
                duration_ms = int(elapsed * 1000)
                setattr(run, f"{step_name}_duration_ms", duration_ms)
                hist = REGISTRY._names_to_collectors.get(
                    "localstock_op_duration_seconds")
                if hist is not None:
                    hist.labels("pipeline", "step", step_name, outcome).observe(elapsed)
    ```
