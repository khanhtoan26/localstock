<task type="checkpoint:human-verify" gate="blocking">
  <name>Task 1: Verify floating sidebar visual appearance and interaction</name>
  <what-built>
Floating collapsible sidebar replacing the old fixed sidebar. Features:
- Always-visible icon rail (~56px) on the left
- Floating overlay panel that slides in/out on icon click (180ms animation)
- Two nav groups: Main (Rankings, Market, Learn) at top, Admin at bottom with separator
- Tooltips on icon hover when collapsed
- No backdrop overlay — toggle via icon click only
- localStorage persistence of collapsed/expanded state
- Content area uses full width minus icon rail (ml-14 instead of ml-60)
  </what-built>
  <how-to-verify>
Start the dev server if not running: `cd apps/helios && npm run dev`

**1. Icon Rail (Collapsed State):**
- Visit http://localhost:3000/rankings
- Verify: narrow icon rail visible on the left (~56px wide)
- Verify: 4 icons visible — 3 at top (Rankings, Market, Learn), 1 at bottom (Admin) with a thin separator line between groups
- Verify: NO logo/branding text in the icon rail
- Verify: NO badge indicators on any icons
- Verify: Content area fills the remaining width (no large empty gap on the left)

**2. Tooltip on Hover:**
- Hover over each icon in the collapsed rail
- Verify: tooltip appears to the RIGHT of the icon after ~200ms delay
- Verify: tooltip shows the page name (e.g., "Rankings", "Market", "Learn", "Admin")
- Verify: tooltip styling is readable (dark bg, light text)

**3. Expand/Collapse Toggle:**
- Click any nav icon (e.g., Rankings)
- Verify: overlay panel slides in from the left with smooth animation (~180ms)
- Verify: panel appears to the right of the icon rail (not replacing it)
- Verify: panel shows "LocalStock" header + "AI Stock Agent" subtitle
- Verify: panel shows full nav links with icon + text label
- Verify: content underneath does NOT shift or move
- Verify: NO backdrop/overlay darkening the content area
- Click the same icon again (or any icon in the rail)
- Verify: panel slides out smoothly back to collapsed state

**4. Navigation + Auto-Collapse:**
- Click an icon to expand the sidebar
- Click a different nav link IN THE EXPANDED PANEL (e.g., "Market")
- Verify: page navigates to that route AND sidebar auto-collapses

**5. Active State:**
- Navigate to /rankings — verify Rankings icon has active styling (slightly different bg)
- Navigate to /market — verify Market icon has active styling
- Check both icon rail and expanded panel show consistent active state

**6. State Persistence (LAY-04):**
- Click an icon to expand the sidebar
- Refresh the page (F5)
- Verify: sidebar is in the EXPANDED state after reload
- Click to collapse it
- Refresh the page again
- Verify: sidebar is COLLAPSED after reload

**7. Theme Compatibility:**
- Toggle to dark mode (theme toggle in header)
- Verify: icon rail and overlay panel use dark theme colors correctly
- Verify: separator, borders, and icon colors are visible and readable
- Toggle back to light mode — verify correct light theme
  </how-to-verify>
