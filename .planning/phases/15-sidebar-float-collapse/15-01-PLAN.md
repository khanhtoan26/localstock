  <action>
Create `apps/helios/src/hooks/use-sidebar-state.ts` with the following implementation:

1. Add `"use client"` directive at top.
2. Import `useState` and `useCallback` from `react`.
3. Define constant `STORAGE_KEY = "localstock-sidebar-collapsed"`.
4. Define constant `DEFAULT_COLLAPSED = true` (per D-09: default collapsed on first visit).
5. Export function `useSidebarState()` that:
   - Uses `useState<boolean>` with a **lazy initializer** function that:
     - Returns `DEFAULT_COLLAPSED` if `typeof window === "undefined"` (SSR safety)
     - Reads `localStorage.getItem(STORAGE_KEY)`
     - Returns `DEFAULT_COLLAPSED` if stored value is `null` (first visit)
     - Returns `stored === "true"` (parse string to boolean)
   - Creates `setCollapsed` via `useCallback` that:
     - Accepts `boolean | ((prev: boolean) => boolean)` parameter
     - Calls `setCollapsedRaw` with a function that resolves the next value
     - Calls `localStorage.setItem(STORAGE_KEY, String(next))` inside the updater
     - Returns the resolved next value
   - Creates `toggle` via `useCallback` that calls `setCollapsed(prev => !prev)`
   - Returns `{ collapsed, setCollapsed, toggle } as const`

**Critical:** Use `useState` lazy initializer (NOT `useEffect`) to read localStorage — this prevents FOUC per Pitfall 1 from RESEARCH.md. The lazy initializer runs synchronously during first render.

**Do NOT use:** `useSyncExternalStore` (overkill — no cross-tab sync needed), Context/Provider (unnecessary — single consumer), Zustand/Jotai (external deps not warranted).

```tsx
function Tooltip({ ...props }: TooltipPrimitive.Root.Props) {
  return <TooltipPrimitive.Root data-slot="tooltip" {...props} />;
}
```

**TooltipTrigger** (Trigger wrapper):
```tsx
function TooltipTrigger({ ...props }: TooltipPrimitive.Trigger.Props) {
  return <TooltipPrimitive.Trigger data-slot="tooltip-trigger" {...props} />;
}
```

**TooltipPortal** (Portal wrapper):
```tsx
function TooltipPortal({ ...props }: TooltipPrimitive.Portal.Props) {
  return <TooltipPrimitive.Portal data-slot="tooltip-portal" {...props} />;
}
```

**TooltipPositioner** (Positioner wrapper with className support):
```tsx
function TooltipPositioner({ className, ...props }: TooltipPrimitive.Positioner.Props) {
  return (
    <TooltipPrimitive.Positioner
      data-slot="tooltip-positioner"
      className={cn("", className)}
      {...props}
    />
  );
}
```

**TooltipContent** (Popup wrapper with styled defaults):
```tsx
function TooltipContent({ className, ...props }: TooltipPrimitive.Popup.Props) {
  return (
    <TooltipPrimitive.Popup
      data-slot="tooltip-content"
      className={cn(
        "rounded-md bg-primary px-3 py-1.5 text-xs text-primary-foreground shadow-md",
        "data-starting-style:opacity-0 data-ending-style:opacity-0",
        "transition-opacity duration-150",
        className
      )}
      {...props}
    />
  );
}
```
