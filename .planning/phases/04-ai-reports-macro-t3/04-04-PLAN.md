1. **Create `src/localstock/services/report_service.py` — ReportService:**
   - Follow the session-based service pattern from ScoringService exactly.
   - Constructor `__init__(self, session: AsyncSession)`:
     ```python
     self.session = session
     self.score_repo = ScoreRepository(session)
     self.report_repo = ReportRepository(session)
     self.macro_repo = MacroRepository(session)
     self.indicator_repo = IndicatorRepository(session)
     self.ratio_repo = RatioRepository(session)
     self.industry_repo = IndustryRepository(session)
     self.sentiment_service = SentimentService(session)
     self.stock_repo = StockRepository(session)
     ```

   - **`async def run_full(self, top_n: int = 20) -> dict`:**
     Pipeline:
     a. Health check Ollama — if down, return early with `{"error": "Ollama not available", "reports_generated": 0}`.
     b. Get top-ranked stocks: `scores = await self.score_repo.get_top_ranked(limit=top_n)`.
     c. Get macro context once (shared across all stocks): `macro_indicators = await self.macro_repo.get_all_latest()`, then `macro_conditions = await MacroCrawler().determine_macro_conditions(macro_indicators)`.
     d. Build macro_data dict for prompt (latest values of each indicator type).
     e. For each scored stock (with per-stock error isolation via try/except):
        - Get technical indicator: `indicator = await self.indicator_repo.get_latest(symbol)`
        - Get financial ratio: `ratio = await self.ratio_repo.get_latest(symbol)`
        - Get sentiment data: `sent_avg = await self.sentiment_service.get_aggregated_sentiment(symbol)`
        - Get stock info: `stock = await self.stock_repo.get_by_symbol(symbol)`
        - Get industry mapping: `mapping = await self.industry_repo.get_stock_mapping(symbol)`
        - Build indicator_data dict from TechnicalIndicator columns + close price from latest StockPrice
        - Compute T+3: `t3_data = predict_3day_trend(indicator_data)`
        - Assemble prompt data: `data = ReportDataBuilder().build(symbol, score, indicator, ratio, sent_avg, macro_data, t3_data, stock_info)`
        - Build prompt: `prompt = build_report_prompt(data)`
        - Generate report: `report = await self.ollama.generate_report(prompt, symbol)`
        - Map recommendation to DB enum: "Mua mạnh" → "strong_buy", "Mua" → "buy", "Nắm giữ" → "hold", "Bán" → "sell", "Bán mạnh" → "strong_sell".
        - Store: `await self.report_repo.upsert({symbol, date, report_type="full", content_json=report.model_dump(), summary=report.summary, recommendation=mapped_rec, t3_prediction=t3_data["direction"], model_used=self.ollama.model, total_score=score.total_score, grade=score.grade})`
     f. Return summary: `{"started_at": ..., "reports_generated": N, "reports_failed": M, "errors": [...]}`

   - **`async def get_reports(self, limit: int = 20) -> list[dict]`:**
     - Get latest reports from DB, return as list of flat dicts with key fields (symbol, date, summary, recommendation, t3_prediction, total_score, grade).

   - **`async def get_report(self, symbol: str) -> dict | None`:**

1. **Create `src/localstock/api/routes/reports.py`:**
   Follow exact pattern from `scores.py` (APIRouter prefix, get_session dependency, asyncio.Lock).
   
   Endpoints:
   - **`GET /api/reports/top`** — Returns latest generated reports for top stocks.
     - Query param: `limit: int = Query(default=20, ge=1, le=100)`
     - Calls `ReportService(session).get_reports(limit)`.
     - Returns `{"reports": [...], "count": N}` or message if no reports.

   - **`GET /api/reports/{symbol}`** — Returns latest report for specific stock.
     - Calls `ReportService(session).get_report(symbol.upper())`.
     - Returns full report dict or 404.

   - **`POST /api/reports/run`** — Triggers report generation for top-ranked stocks.
     - Query param: `top_n: int = Query(default=20, ge=1, le=50)`
     - Uses asyncio.Lock (report_lock) to prevent concurrent generation.
     - Returns 409 if already running.
     - Calls `ReportService(session).run_full(top_n)`.

2. **Create `src/localstock/api/routes/macro.py`:**
   Follow the same pattern.
   
   Endpoints:
   - **`GET /api/macro/latest`** — Returns latest macro indicators.
     - Calls `MacroRepository(session).get_all_latest()`.
     - Returns `{"indicators": [{indicator_type, value, period, source, trend, recorded_at}, ...]}`.

   - **`POST /api/macro`** — Manual entry for macro indicators (interest rate, CPI, GDP).
     - Request body validated with Pydantic:
       ```python
       class MacroInput(BaseModel):
           indicator_type: str = Field(pattern="^(interest_rate|exchange_rate_usd_vnd|cpi|gdp)$")
           value: float
           period: str = Field(min_length=4, max_length=20)  # e.g., "2026-Q1", "2026-04"
           source: str = Field(default="manual", max_length=50)
       ```
     - Creates MacroIndicator row via MacroRepository.bulk_upsert with trend computed from previous value.
     - Returns `{"status": "ok", "indicator": {...}}`.

   - **`POST /api/macro/fetch-exchange-rate`** — Triggers exchange rate fetch from VCB.
     - Calls `MacroCrawler().fetch_exchange_rate()`.
     - If result: stores via MacroRepository.bulk_upsert.
     - Returns `{"status": "ok", "rate": {...}}` or `{"status": "failed", "error": "..."}`.
