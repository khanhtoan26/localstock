1. **Create `src/localstock/reports/t3.py` — T+3 prediction:**
   - Implement `predict_3day_trend(indicator_data: dict) -> dict` per research §Pattern 3.
   - Input dict keys (from TechnicalIndicator columns + close from StockPrice): `rsi_14`, `macd_histogram`, `trend_direction`, `trend_strength`, `nearest_support`, `nearest_resistance`, `close`, `relative_volume`, `volume_trend`.
   - Signal aggregation logic (each contributes +1 or -1):
     a. **RSI momentum**: 30 < rsi < 50 → +1 (recovering), rsi > 70 → -1 (overbought), else 0.
     b. **MACD histogram**: macd_h > 0 → +1, macd_h < 0 → -1.
     c. **Trend + strength**: uptrend AND adx > 25 → +1, downtrend AND adx > 25 → -1.
     d. **Support/resistance ratio**: upside > downside*2 → +1, downside > upside*2 → -1.
     e. **Volume confirmation**: relative_volume > 1.5 AND trend=uptrend → +1.
   - Direction: signals >= 2 → "bullish", signals <= -2 → "bearish", else "neutral".
   - Confidence: |signals| >= 3 → "high", |signals| >= 2 → "medium", else "low".
   - Reasons: list of Vietnamese strings explaining each signal (per research examples).
   - T+3 warning: always include the standard warning text: `"⚠️ CẢNH BÁO T+3: Cổ phiếu mua hôm nay chỉ có thể bán sau 3 ngày làm việc. Hãy đảm bảo xu hướng đủ mạnh để giữ vị thế trong ít nhất 3 phiên giao dịch."`.
   - Handle None values: skip signal if required field is None. If all None → neutral, low, empty reasons.
