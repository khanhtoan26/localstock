Create `apps/helios/src/lib/glossary-linker.ts` with these exact exports:

1. **Types:**
   ```typescript
   export interface AliasMapping {
     aliasLower: string;  // Lowercased alias for case-insensitive matching
     alias: string;       // Original alias text
     entry: GlossaryEntry;
   }

   export interface GlossaryMatch {
     entry: GlossaryEntry;
     matchedText: string; // Original text from source (preserves casing)
   }

   export type TextSegment = string | GlossaryMatch;
   ```

2. **`buildAliasMap()`** — no parameters. Calls `getAllEntries()` internally. Iterates all entries and their aliases, creates `AliasMapping[]`, sorts by `aliasLower.length` descending (per D-05 longest-first). Returns the sorted array.

3. **`isWordBoundary(char: string | undefined): boolean`** — returns `true` if `char` is `undefined` (start/end of string) or does NOT match `/[\p{L}\p{N}]/u` (not a Unicode letter or digit). This prevents "MA" from matching inside "thematic". Export this for testing.

4. **`scanText(text, aliasMap, linkedIds)`** — the core scanner:
   - Takes `text: string`, `aliasMap: AliasMapping[]`, `linkedIds: Set<string>`
   - Lowercases `text` once to `textLower` for comparison
   - Iterates character-by-character using a `cursor` position
   - At each cursor position, checks all aliases in aliasMap order (longest first):
     - Skip if `linkedIds.has(entry.id)` (first-occurrence-only per D-07)
     - Check if `textLower.startsWith(aliasLower, cursor)` (replaces indexOf at cursor — simpler)
     - If potential match found, check word boundaries: `isWordBoundary(text[cursor - 1])` and `isWordBoundary(text[cursor + aliasLower.length])`
     - On match: push preceding text segment (if any), push `GlossaryMatch`, add `entry.id` to `linkedIds`, advance cursor past match, break inner loop
   - If no alias matches at current position, find next potential match position by checking `textLower.indexOf(aliasLower, cursor + 1)` for all aliases, take minimum. Push text up to that point, advance cursor. If no future match exists, push remaining text and break.
   - Filter out empty string segments from result before returning

**Important constraints (per D-06):** Matching is case-insensitive via `toLowerCase()` — do NOT apply any diacritic normalization. Vietnamese diacritics in aliases must match exactly.

**Do NOT modify `glossary.ts`** — this module is a consumer only.
