### Article Content Typography (via `@tailwindcss/typography` prose)

Expanded entry content renders via `react-markdown` inside a `prose` container. The `prose` class inherits theme colors:

| Element | Prose Default | Override |
|---------|---------------|----------|
| Prose body | 16px / 1.75 line-height | `prose-sm` (14px / 1.71) to match app body density |
| Prose headings (h3) | 20px / 600 weight | No override — default is appropriate |
| Prose paragraphs | 16px / 400 | `prose-sm` reduces to 14px |
| Prose code | 14px mono | Apply `prose-invert` conditionally for dark mode |
| Prose strong | inherit / 600 | No override |
| Max width | 65ch | `max-w-none` — card already constrains width |

Apply: `prose prose-sm dark:prose-invert max-w-none prose-headings:text-foreground prose-p:text-foreground/90 prose-strong:text-foreground prose-code:text-muted-foreground`

## Page Structure & Layout

### Hub Page (`/learn`)

**Focal point:** The 3-column category card grid — each card's icon and Vietnamese title draw the eye and invite exploration.

```
┌─────────────────────────────────────────┐
│ Display: "Học — Kiến Thức Đầu Tư"      │
│ Body: subtitle description              │
├─────────────────────────────────────────┤
│ ┌───────────┐ ┌───────────┐ ┌─────────┐│
│ │ Technical │ │Fundamental│ │  Macro  ││
│ │ Card      │ │ Card      │ │  Card   ││
│ │ icon      │ │ icon      │ │  icon   ││
│ │ count     │ │ count     │ │  count  ││
│ │ desc      │ │ desc      │ │  desc   ││
│ └───────────┘ └───────────┘ └─────────┘│
└─────────────────────────────────────────┘
```

- 3-column grid on desktop (`grid-cols-3 gap-6`), 1-column on mobile (`grid-cols-1`)
- Each category card is a `<Link>` wrapping a `<Card>` — clickable to navigate to `/learn/[category]`
- Card contains: category icon (32×32px, `text-primary`), Vietnamese title (18px semibold), English subtitle (12px muted), entry count badge, 1-line description

### Category Page (`/learn/[category]`)

**Focal point:** The search input at the top followed by the vertically stacked glossary entry cards — the search bar anchors user attention and the entry list is the primary content.

```
┌─────────────────────────────────────────┐
│ Heading: "Chỉ Báo Kỹ Thuật"           │
│ Badge: "Kỹ Thuật" + entry count        │
├─────────────────────────────────────────┤
│ ┌─────────────────────────────────────┐ │
│ │ 🔍 Search: "Tìm chỉ báo..."       │ │
│ └─────────────────────────────────────┘ │
├─────────────────────────────────────────┤
│ ┌─────────────────────────────────────┐ │
│ │ ▶ RSI — Chỉ Số Sức Mạnh Tương Đối │ │
│ │   Short definition text...          │ │
│ ├─────────────────────────────────────┤ │
│ │ ▶ MACD — Hội Tụ Phân Kỳ...        │ │
│ │   Short definition text...          │ │
│ ├─────────────────────────────────────┤ │
│ │ ▼ SMA — Đường Trung Bình Đơn Giản │ │
│ │   Short definition text...          │ │
│ │ ┌─────────────────────────────────┐ │ │
│ │ │ Full markdown article content   │ │ │
│ │ │ with formula, interpretation,   │ │ │
│ │ │ examples, notes...              │ │ │
│ │ └─────────────────────────────────┘ │ │
│ └─────────────────────────────────────┘ │
└─────────────────────────────────────────┘
```

- Search input at top: full-width, 40px height, Search icon (16px) left-inset, placeholder text
- Entry cards stacked vertically with 16px gap
- Each entry card: `Collapsible` wrapping `Card`
- Collapsed state: shows term (Vietnamese + English in parentheses) as CardTitle + shortDef as CardDescription
- Expanded state: adds full markdown content in CardContent with `prose prose-sm` styling
- ChevronDown/ChevronUp icon (16px) on the right of card header indicates state
- Formula block (if present): rendered in a `<code>` block with `bg-muted rounded-md p-3 font-mono text-sm` styling

## Interaction Contract

### Navigation

| Interaction | Behavior |
|-------------|----------|
| Click sidebar "Học" | Navigate to `/learn` hub page |
| Click category card on hub | Navigate to `/learn/[category]` |
| Browser back from category | Return to `/learn` hub |
| Invalid category slug | Next.js `notFound()` — 404 page |

### Search & Filter

| Property | Value |
|----------|-------|
| Trigger | `onChange` on input — real-time filtering as user types |
| Debounce | None (client-side filter on ~25 entries is instant) |
| Match fields | `term`, `termEn`, `shortDef`, `aliases[]` — search across all text fields |
| Match algorithm | Normalize both query and field: `str.normalize('NFD').replace(/[\u0300-\u036f]/g, '').toLowerCase()` |
| Clear | "×" button appears when input has value — click resets to empty, shows all entries |
| Result count | Label below search: "{N} kết quả" (updates reactively) |
| No results | Inline message: "Không tìm thấy kết quả cho "{query}"" below search bar |
| Keyboard | Escape key clears search input and restores focus |

### Expand/Collapse Entry Cards

| Property | Value |
|----------|-------|
| Trigger | Click anywhere on card header (title + shortDef area) |
| Animation | `@base-ui/react` Collapsible default height transition |
| Multiple open | Allowed — multiple entries can be expanded simultaneously |
| URL anchors | Each entry card has `id={entry.id}` — supports `/learn/technical#rsi` deep links from Phase 10 |
| Auto-expand on anchor | If URL contains `#hash`, auto-expand matching entry and scroll into view on mount |
| Cursor | `cursor-pointer` on CollapsibleTrigger |

### Hub Page Category Cards

| Property | Value |
|----------|-------|
| Hover state | `hover:ring-2 hover:ring-primary/20` + subtle scale: `hover:shadow-md transition-shadow` |
| Click | Full card is clickable link — no separate button |
| Focus | `focus-visible:ring-2 focus-visible:ring-ring` (keyboard navigation) |

## Glossary Data Model Contract

Source: CONTEXT.md D-04, D-05, D-06

```typescript
// src/lib/glossary.ts

export type GlossaryCategory = "technical" | "fundamental" | "macro";

export interface GlossaryEntry {
  id: string;                    // URL-safe slug: "rsi", "pe-ratio", "cpi"
  term: string;                  // Vietnamese name: "Chỉ số sức mạnh tương đối (RSI)"
  termEn: string;                // English name: "Relative Strength Index"
  aliases: string[];             // Phase 10 matching: ["RSI", "chỉ số RSI", "Relative Strength Index"]
  category: GlossaryCategory;
  shortDef: string;              // 1-sentence Vietnamese definition
  content: string;               // Full markdown article (Vietnamese, English terms in parens)
  formula?: string;              // Optional: LaTeX-free plain text formula
}

// Export as typed Record for O(1) lookup by id
export const glossary: Record<string, GlossaryEntry> = { ... };

// Helper: get entries by category (used by category pages)
export function getEntriesByCategory(category: GlossaryCategory): GlossaryEntry[];

// Helper: get all entries (used by hub page for counts)
export function getAllEntries(): GlossaryEntry[];

// Helper: diacritic-insensitive search
export function normalizeForSearch(text: string): string;
```

Minimum entry count: 25 entries total, distributed across categories:
- Technical: ~10 entries (RSI, MACD, SMA, EMA, Bollinger Bands, OBV, VWAP, Stochastic, ADX, ATR)
- Fundamental: ~10 entries (P/E, P/B, EPS, ROE, ROA, D/E, Revenue Growth, Profit Growth, Market Cap, Current Ratio)
- Macro: ~5 entries (CPI, GDP Growth, Interest Rate, Exchange Rate, Money Supply M2)
